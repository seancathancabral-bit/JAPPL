Welcome to JAPPL

JAPPL is a lightweight, syntax-tolerant programming language built inside Scratch. The goal is to make coding extremely easy for beginners without losing flexibility.

JAPPL 4.1.5 Stable

What Makes JAPPL Special?

  • Very easy to learn
  • Syntax-tolerant
  • Variable system + operators + loops
  • Editable text engine
  • Fast compiler → executer structure
